This form will take a github access token and create a fork of /mmkayz/hjoy-forker into the GitHub account of the user who owns the access token.

To create an access token:

  • Log in to Github https://github.com
  • Navigate to Settings -> Developer Settings -> Personal access tokens
  • Under Tokens(classic). Generate a new token
  • Copy and paste the token into the input below to create a fork of the repository.
